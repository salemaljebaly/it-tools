# IT Tools

A small collection of lightweight scripts and “manager/dev helper” tools.

## Tools

- `tools/gmail-ack-reply-bot/` — Gmail auto-acknowledgement reply bot (Google Apps Script)

## Add A Tool

1. Create a folder: `tools/<tool-name>/`
2. Add a short `README.md` with setup + usage
3. Add a single bullet entry under “Tools” above
